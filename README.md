# Gitignore
 Файл .gitignore для ваших локальных и не только репозитроиев для VS C++

### Как пользоваться?
 1) Качаете репозиторий в `.zip` или с гитхаба напрямую
 2) Берёте файл ``.gitignore`` и кидаете в новосозданный вами репозиторий
 3) Делаете ``commit``
 4) Готово!!!
